# Employee Holiday System

# goal 
Build a simple leave management system with Flask

# stack
- python (flask)
- mySQL (later)

# Roles:
- Employees: request leave
- Admin:approve/reject request leave

# Rules
- Each employee has 15 days off per year rather than the offical holiday
- Weekends (Saturday-Sunday) are not counted
- Leave Request are started by 'pending'
